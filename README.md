# goatCSV
Python script to get items sold on GOAT App 
![code](images/goat.gif)
![cvs](/images/image.png)

## How to:
1. pip install -r req.txt
2. python goat.py
3. enter log in email/userid and password
4. check folder for saved files: goat.json and goat.csv
